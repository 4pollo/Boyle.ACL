System.Data.ToSQL 方法
根据指定条件生成SQL查询语句
语法
System.Data.ToSQL(Collection, Condition, Order)
参数
Collection Array(数组)
  必选项，不能为空。包括表，字段以及取的记录数。格式必须遵守 Array("表名", "字段名1,字段名2...", "数量")的形式。
Condition String(字符串) 或 Array(数组)
  必选项，可以为空。查询的条件。可以直接写条件语句。如果将多个条件用AND进行连接的话，也可以写成数组形式。格式必须遵守 Array(字段1=值1, 字段2>值2, ...) 的形式。
Order String(字符串)
  必选项，可以为空。排序的方式。
返回值
String(字符串)
注释
无
示例
 1.With System.Data
 2.    Dim strSQL1, strSQL2, strSQL3, strSQL4
 3.    '// 取出USER表中包括全部字段的所有记录
 4.    strSQL1 = .ToSQL(Array("USER", "", ""), "", "")
 5.    '// 取出USER表中包括ID,NAME,PASS三个字段，条件为NAME等于Boyle或PASS等于123456的所有记录
 6.    strSQL2 = .ToSQL(Array("USER", "ID,NAME,PASS", 0), "NAME='Boyle' Or PASS='123456'", "")
 7.    '// 取出USER表中包括ID,NAME两个字段，并对ID列进行倒序排序之后的前5条记录
 8.    strSQL3 = .ToSQL(Array("USER", "ID,NAME", 5), "", "ID DESC")
 9.    '// 取出USER表中包括全部字段，条件为NAME列不为空并密码为123456时，并对ID列进行倒序排序之后的前5条记录
10.    strSQL4 = .ToSQL(Array("USER", "", 5), Array("[NAME]<>''", "PASS='123456'"), "[ID] DESC")
11.    '// 输出SQL语句
12.    System.W strSQL1 & strSQL2 & strSQL3 & vbCrLf & strSQL4
13.End With
14.'// 最终输出结果：
15.strSQL1 = Select * From [USER]
16.strSQL2 = Select [ID],[NAME],[PASS] From [USER] Where NAME='Boyle' Or PASS='123456'
17.strSQL3 = Select Top 5 [ID],[NAME] From [USER] Order By ID DESC
18.strSQL4 = Select Top 5 * From [USER] Where [NAME]<>'' AND PASS='123456' Order By [ID] DESC
