System.Array.HasIndex 方法
检测是否包含某个下标
语法
System.Array.HasIndex index
参数
index Integer (整数) 或 String (字符串)
  可以使用数字下标检测，也可以使用字符串下标检测。
返回值
Boolean (布尔值)
注释
使用该方法可以检测数组中是否包含某个元素，如果是Hash数组的话，检测的目标仅是列值，不能检测列名。
示例
 1.'// 创建一个新对象
 2.Dim ArrayX: Set ArrayX = System.Array.New
 3.ArrayX.Hash = "CHN:中国 HK:香港"
 4.System.WB ArrayX.HasIndex("中国")
 5.System.WB ArrayX.HasIndex("HK")
 6.'// 最终输出结果：
 7.Flase
 8.True
