System.Array.Hash 属性
设置源数组为Hash数组并赋值，此属性可读可写
语法
System.Array.Hash = Value
参数
Value String(字符串) 或 Array(数组)
  可选。可以是数组，也可以是以空格分隔的字符串。数组的每个元素或者字符串的每个成员，都必须用冒号(:)分隔，冒号左边的部分就是Hash的列名（即字符串下标）。
返回值
Array(数组)
注释
创建一个对象后，通过此方法赋值，该对象就是Hash数组。
示例
 1.Dim ArrayX
 2.'// 创建一个对象
 3.Set ArrayX = System.Array.New
 4.'// 通过以空格分隔的字符串赋值
 5.ArrayX.Hash = "A:中 B:华 C:人 D:民 E:共 F:和 G:国"
 6.'// 通过数组赋值
 7.'// ArrayX.Hash = Array("A:中", "B:华", "C:人", "D:民", "E:共", "F:和", "G:国")
 8.'// 取出为普通数组
 9.Dim arr: arr = ArrayX.Hash
10.Dim I: For I = 0 To UBound(arr)
11.    System.WB "arr("&I&")的值为：" & arr(I)
12.Next
13.'// 最终输出结果：
14.arr(0)的值为： A:中
15.arr(1)的值为： B:华
16.arr(2)的值为： C:人
17.arr(3)的值为： D:民
18.arr(4)的值为： E:共
19.arr(5)的值为： F:和
20.arr(6)的值为： G:国
