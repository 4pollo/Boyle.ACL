System.Array 概述
处理数组(Array)和哈希表(Hash)的类
属性
名称 说明
Array.IgnoreCase 设置和查询数组操作时是否忽略数组值的大小写，此属性可读可写
Array.Data 设置源数组为简单数组并赋值，此属性可读可写
Array.Hash 设置源数组为Hash数组并赋值，此属性可读可写
Array.First 返回数组第一个元素的值，此属性只读
Array.Last 返回数组最后一个元素的值，此属性只读
Array.End 返回数组的最大数字下标，此属性只读
Array.Min 返回数组最小元素的值，此属性只读
Array.Max 返回数组最大元素的值，此属性只读
Array.Size 返回当前数组的长度（元素个数），此属性只读
Array.Maps 将Hash数组映射为 Dictionary 对象，此属性可读可写
Array.Serialize 序列化Hash数组，此属性只读
方法
名称 说明
Array.At 根据下标设置或获取数组某一项的值
Array.Has 检测数组中是否包含某个元素
Array.HasIndex 检测是否包含某个下标
Array.Index 根据Hash的列名获取其数字下标
Array.IndexOf 检测元素在数组中的数字下标
Array.IndexOfHash 检测元素在Hash数组中的字符串下标
Array.Delete 删除指定下标的数组元素
Array.Delete_ 删除指定下标的数组元素，返回新的对象
Array.Insert 向指定数字下标处插入一个或多个元素
Array.Insert_ 向指定数字下标处插入一个或多个元素，返回新的对象
Array.Pop 删除数组的最后一个元素
Array.Pop_ 删除数组的最后一个元素，返回新的对象
Array.Push 添加一个元素到数组的末尾
Array.Push_ 添加一个元素到数组的末尾，返回新的对象
Array.Shift 删除数组的第一个元素
Array.Shift_ 删除数组的第一个元素，返回新的对象
Array.UnShift 添加一个或多个元素到数组的开头
Array.UnShift_ 添加一个或多个元素到数组的开头，返回新的对象
