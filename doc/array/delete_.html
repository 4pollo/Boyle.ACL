System.Array.Delete_ 方法
删除指定下标的数组元素，返回新的对象
语法
System.Array.Delete_ index
参数
index Integer (整数) 或 String (字符串)
  被删除元素的下标，可以是数字，抑或是Hash的列名。同时删除多个元素时，还可以指定下标的区间，用 - 表示区间，\s 表示开头，\e 表示结尾。
返回值
Object (对象)
注释
使用此方法可以删除指定下标的数组元素，可以只删除一个，也可以同时删除多个。index可以是自然数，也可以是字符串，还可以是数字区间。
使用该方法，返回的是删除元素后的新数组对象，不改变原数组的数据。
示例
 1.'// 创建一个新对象
 2.Dim ArrayX: Set ArrayX = System.Array.New
 3.ArrayX.Hash = "CN:中国 HK:香港 MO:澳门 TW:台湾 MY:马来西亚 SG:新加坡 DE:德国 JP:日本 US:美国"
 4.'// 通过数字下标删除一个元素
 5.System.WB "删除数字下标为1的元素后：" & ArrayX.Delete_(1).ToString
 6.'// 通过字符串下标删除两个元素
 7.System.WB "删除字符串下标为""DE""和""JP""的元素后 ：" & ArrayX.Delete_("DE.JP").ToString
 8.System.WB "此时原数组的长度为：" & ArrayX.Size & "，最大数组下标为：" & ArrayX.End
 9.'// 通过区间删除下标为0到2，以及下标为5的四个元素（用 - 表示区间, \s 表示开头, \e 表示结尾）
10.System.WB "删除下标为""开头-2""和""5""的元素后：" & ArrayX.Delete_("\s-2,5").ToString
11.'// 最终输出结果：
12.删除数字下标为1的元素后：中国,澳门,台湾,马来西亚,新加坡,德国,日本,美国
13.删除字符串下标为"DE"和"JP"的元素后 ：中国,香港,澳门,台湾,马来西亚,新加坡,美国
14.此时原数组的长度为：9，最大数组下标为：8
15.删除下标为"开头-2"和"5"的元素后：台湾,马来西亚,德国,日本,美国
