System.Array.Maps 属性
将Hash数组映射为 Dictionary 对象，此属性可读可写
语法
System.Array.Maps = hash
参数
hash Array(数组)
  可选。目标数组。
返回值
Object (对象)
注释
转换为 Dictionary 对象后，可对其使用 Dictionary 对象自有的属性和方法。
