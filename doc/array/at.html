System.Array.At 方法
根据下标设置或获取数组某一项的值
语法
System.Array.At(index)[ = value]
参数
index Integer (整数) 或 String (字符串)
  数组的下标，只能由自然数、字母、下划线(_)、点(.)和斜杠(/)组成。
数组是普通数组时，下标就是自然数；数组是Hash数组时，下标除了可以是自然数之外，还可以是非空字符串。
value Array (数组) 或 Boolean (布尔值) 或 Datetime (时间日期) 或 Numeric (数值) 或 String (字符串)
  被设置的数组的值
返回值
Array (数组) 或 Boolean (布尔值) 或 Datetime (时间日期) 或 Numeric (数值) 或 String (字符串)
注释
无
示例
 1.'// 创建一个新对象
 2.Dim ArrayX: Set ArrayX = System.Array.New
 3.ArrayX.Data = "中 华 人 民 共 和 国"
 4.'// 设置下标为5的数组值为“产”
 5.ArrayX.At(5) = "产"
 6.'// 设置下标为6的数组值为“党”，下标为7的数组值为“万岁”
 7.'// 此时数组已经变成：Array("中", "华", "人", "民", "共", "产", "党", "万岁")
 8.ArrayX(6) = "党": ArrayX(7) = "万岁"
 9.'// 输出下标为7的数组值，页面将输出：万岁
10.System.WB ArrayX(7)
11.'// 使用字符串做为下标来赋值和取值
12.ArrayX("NOTE") = "万万岁！"
13.System.WB ArrayX("NOTE")
14.'// 仍然可以用数字下标获取数组中的值
15.'// System.WB ArrayX(8)
16.'// 页面将输出：万万岁！
17.Dim arr: arr = ArrayX.Data
18.Dim I: For I=0 To UBound(arr)
19.    System.W arr(I)
20.Next
21.'// 最终输出结果：
22.万岁
23.万万岁！
24.中华人民共产党万岁万万岁！
